ใส่ URL Google Apps Script เพื่อเชื่อมกับฐานข้อมูล Google Sheets

ขั้นตอนตั้งค่า (ใช้ไฟล์ nahla_apps_script_v2.gs):
1. เปิด Google Sheet → Extensions → Apps Script
2. ลบโค้ดเดิม → วางโค้ดจาก nahla_apps_script_v2.gs
3. รัน initSheets() ก่อน 1 ครั้ง
4. Deploy → New deployment → Web app
5. Execute as: Me · Access: Anyone
6. Copy URL → วางด้านบน
⚠ ต้อง Deploy ใหม่ทุกครั้งที่แก้โค้ด
